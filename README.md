# Unemployment Analysis with Python

## Oasis Infobyte Data Science Internship - Task 2

## 📌 Project Overview

This project analyzes unemployment trends in India using Python. The objective is to explore unemployment data, identify patterns, visualize trends, and understand the impact of the COVID-19 pandemic on unemployment rates.

---

## 🎯 Objectives

- Analyze unemployment data using Python.
- Perform Exploratory Data Analysis (EDA).
- Visualize unemployment trends across different states.
- Compare unemployment rates before and after COVID-19.
- Understand relationships between employment-related variables.

---

## 📂 Dataset

The project uses the following datasets:

- `Unemployment in India.csv`
- `Unemployment_Rate_upto_11_2020.csv`

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Project Workflow

1. Import required libraries.
2. Load the dataset.
3. Explore the dataset.
4. Check and handle missing values.
5. Convert the Date column into datetime format.
6. Perform descriptive statistics.
7. Create visualizations.
8. Compare unemployment before and after COVID-19.
9. Summarize the findings.

---

## 📈 Visualizations

- Region-wise Average Unemployment Rate (Bar Chart)
- Time Series Analysis
- Correlation Heatmap
- Pre-COVID vs Post-COVID Comparison

---

## 🔍 Key Findings

- Unemployment rates vary across different states.
- COVID-19 significantly increased unemployment.
- Time-series analysis shows noticeable changes during the lockdown period.
- Correlation analysis helps understand relationships between employment indicators.
